# Admin Dashboard

This project is part of The Odin Project curriculum.

It is a static Admin Dashboard layout built using HTML and CSS.  
The layout is structured with CSS Grid and Flexbox and includes:

- Sidebar navigation
- Header with search and user info
- Projects section
- Announcements section
- Trending section

The goal of this project was to practice advanced layout techniques,
component structuring, and visual hierarchy.

## Technologies Used

- HTML5
- CSS3
- CSS Grid
- Flexbox

## Status

Static layout (not responsive).
